# ⚡ Créer un Compte Railway en 2 Minutes

## 🎯 Étapes Ultra-Rapides

### 1. Créer un Compte Railroad.app
```
1. Allez sur: https://railway.app
2. Cliquez "Start Free" (bouton en haut à droite)
3. Connectez-vous avec GitHub (plus simple):
   - Cliquez "Continue with GitHub"
   - Autorisez l'accès
3. Validez votre email
```

**C'est tout!** Vous avez maintenant:
- ✅ Compte Railway
- ✅ $5 de crédit gratuit
- ✅ Accès au dashboard

---

### 2. Connecter Votre Repo GitHub

```
1. Railway Dashboard: https://railway.app/dashboard
2. Cliquez "New Project"
3. Choisissez "Import from GitHub"
4. Sélectionnez: collecte-taxe
5. Railway configure tout automatiquement
```

---

### 3. C'est Fini!

Attendez 5-10 minutes que Railway:
✅ Clone le repo  
✅ Build le Dockerfile  
✅ Lance les conteneurs  
✅ Crée PostgreSQL  

Voilà! Votre app est live 🚀
