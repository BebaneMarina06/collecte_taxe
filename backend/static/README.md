# Dossier Static

Ce dossier contient les fichiers statiques utilisés par l'application, notamment :

## Logo

Placez le logo de la Mairie de Libreville dans ce dossier avec le nom `logo.png`.

Le logo sera automatiquement inclus dans les exports PDF des rapports.

### Formats supportés
- PNG (recommandé)
- JPG/JPEG

### Taille recommandée
- Largeur : 200-400 pixels
- Hauteur : 200-400 pixels
- Format : PNG avec transparence (optionnel)
